# Шаг 6: Telegram-бот для уведомлений

Создай файл `scripts/telegram_bot.py`.

## Что делает скрипт

Два режима работы:
1. **Мониторинг** — каждые 60 секунд проверяет новые заказы в RetailCRM, уведомляет о крупных (> 50 000 KZT)
2. **Команды** — отвечает на команды пользователя: /stats, /report

## Режим мониторинга (фоновый)

1. Каждые 60 секунд запрашивать заказы из RetailCRM с фильтром за последний час
2. Для каждого заказа вычислить сумму: `sum(item.quantity * item.initialPrice)`
3. Если сумма > 50 000 KZT и ID заказа ещё не отправлялся — отправить уведомление
4. Хранить отправленные ID в set (в памяти)

Формат уведомления (HTML parse_mode):
```
<b>Новый заказ #НОМЕР</b>
Сумма: <b>СУММА KZT</b>
Клиент: ИМЯ ФАМИЛИЯ
Город: ГОРОД
```

## Команды

### /stats
Забирает данные из Supabase и отправляет сводку:
```
<b>Статистика Nova</b>

Заказов: 50
Общая сумма: 1 234 567 KZT
Средний чек: 24 691 KZT
Топ город: Алматы (41)
Топ источник: Instagram (19)
```

### /report
Забирает все инсайты из таблицы `insights` в Supabase и отправляет одним сообщением:
```
<b>AI-отчёт</b>
<i>Сгенерировано: ДАТА</i>

ТЕКСТ_РЕКОМЕНДАЦИЙ
```

Если таблица `insights` пуста — ответить: "Отчёт ещё не сгенерирован. Запустите generate_insights.py"

## Архитектура

Использовать polling через Telegram Bot API (`getUpdates`) в одном цикле с мониторингом:
- Основной цикл: каждые 3 секунды проверять новые сообщения через `getUpdates`
- Каждые 60 секунд (по таймеру) — проверять заказы в RetailCRM

Всё через `requests`, без доп. библиотек для Telegram.

## Credentials

Читать из `.env`:
- `RETAILCRM_URL`
- `RETAILCRM_API_KEY`
- `SUPABASE_URL`
- `SUPABASE_KEY`
- `TELEGRAM_TOKEN`
- `TELEGRAM_CHAT_ID`

## Требования к коду

- Type hints на всех функциях
- Докстринги на русском (описание + Args + Returns)
- Комментарии только там где неочевидно
- Обработка ошибок через try/except — при ошибке вывести и продолжить цикл
- Никаких эмодзи
- При запуске вывести: `Бот запущен. Проверяю заказы каждые 60 секунд...`
- Порядок: импорты -> константы -> функции -> точка входа
