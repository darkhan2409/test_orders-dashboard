# Шаг 5: Дашборд

Создай файл `dashboard/index.html` — одностраничный дашборд с графиками.

## Технологии

- Supabase JS — подключить через CDN
- Chart.js — подключить через CDN
- Чистый HTML/CSS/JS, без фреймворков
- `SUPABASE_URL` и `SUPABASE_ANON_KEY` захардкодить в файле (заполню вручную)

## Данные

Дашборд читает из Supabase три таблицы:
- `orders` — поля: id, number, status, total, created_at, customer_name, city, utm_source
- `order_items` — поля: order_id, product_name, quantity, price
- `insights` — поля: id, content, generated_at

## Компоненты

### 3 карточки вверху

1. **Всего заказов** — количество записей в orders
2. **Общая сумма** — sum(total), форматировать с пробелами: "1 234 567 KZT"
3. **Средний чек** — avg(total), форматировать так же

Числа в карточках должны анимироваться при загрузке (count-up эффект от 0 до значения за ~1 секунду). Реализовать через requestAnimationFrame, без библиотек.

### Фильтры

Под карточками — строка фильтров:
- **Период**: кнопки "7 дней" / "30 дней" / "Все время" (по умолчанию "Все время")
- **Город**: выпадающий список, заполняется из данных (+ "Все города")

При смене фильтра — все карточки и графики перерисовываются с учётом фильтра.

### 4 графика

1. **Заказы по дням** (line chart) — ось X: дата (created_at), ось Y: количество заказов
2. **Города** (doughnut chart) — распределение заказов по полю city
3. **Источники трафика** (bar chart) — распределение по utm_source
4. **Топ товаров по выручке** (horizontal bar chart) — из order_items: sum(quantity * price) группировка по product_name

### Блок "AI-аналитика"

Под графиками — отдельный блок:
- Заголовок: "AI-аналитика"
- Подзаголовок мелким текстом: дата генерации (из поля generated_at)
- Содержимое: список рекомендаций из таблицы `insights`
- Каждая рекомендация — в отдельной карточке
- Если таблица `insights` пуста — показать: "Запустите generate_insights.py для генерации AI-рекомендаций"

### Кнопка "Скачать CSV"

В правом верхнем углу — кнопка, которая скачивает текущие (отфильтрованные) заказы в CSV-файл. Формат: id, number, status, total, created_at, customer_name, city, utm_source.

## Дизайн

- Тёмная тема (тёмный фон #0f172a, карточки #1e293b, текст #e2e8f0)
- Минималистичный, современный стиль
- Адаптивная верстка: 2 графика в ряд на десктопе, 1 на мобильном (CSS grid)
- Заголовок страницы: "Nova — Аналитика заказов"
- Шрифт: Inter (через Google Fonts) с fallback на system-ui
- Кнопки фильтров: при активной — яркий акцентный цвет (#3b82f6)
- Карточки AI-инсайтов: лёгкая обводка с градиентом, чтобы выделялись

## Требования

- Всё в одном HTML-файле (inline CSS и JS)
- Никаких эмодзи
- При ошибке загрузки данных — показать сообщение пользователю
